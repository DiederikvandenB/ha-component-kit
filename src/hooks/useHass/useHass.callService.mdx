# callService()
The call service helper will allow you to trigger any action in home assistant.

If you have the developer tools enabled in home assistant, click on this, then services.

Here you'll see a dropdown list, if you open this you'll see there's a "sub title" under each dropdown option, eg "climate.turn_on", in this particular example, "climate" is the domain and "turn_on" is the service.

This is an easy way to visualize all the available domains/service for your home assistant instance.

### Definition
<Source language="ts" code={`callService({
  domain: string; // "climate", "light" etc
  service: string; // "turn_on", "toggle" etc
  serviceData?: object; // any data to pass to the request
  target?: {
    entity_id?: string | string[];
    device_id?: string | string[];
    area_id?: string | string[];
  }
})`} />

### Example Usage

```ts
function CallServiceExample() {
  const { callService } = useHass();
  const turnOnAC = useCallback(() => {
    callService({
      domain: 'climate',
      service: 'turn_on',
      target: {
        entity_id: 'climate.air_conditioner'
      }
    })
  }, [callService]);
